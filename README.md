# 🏦 Sistema Bancário (Java)

Sistema bancário simples desenvolvido em Java utilizando conceitos de Programação Orientada a Objetos (POO).  
A aplicação roda no terminal e simula operações básicas de um banco.

---

## 🚀 Funcionalidades

- 💵 Depósito de valores
- 💸 Saque de valores
- 🔐 Validação de PIN com limite de tentativas
- ⚠️ Validação de entrada (apenas números)
- 💰 Exibição de saldo
- 🧾 Exibição de extrato simples

---

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Tratamento de Exceções
- Scanner (entrada de dados)
